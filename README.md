# 济图通 (Jitutong) — 项目说明

## 项目简介
济图通是一套面向校内的交互式地图与校园 Wiki 平台，集地图浏览、地点信息展示、用户社区、搜索与导航、以及后台管理于一体，支持管理员对地点权威信息的发布与审核。

## 主要功能
- 交互式地图
  - 地图浏览：支持缩放、拖拽等手势操作，展示校区建筑、道路、景观。
  - 查看地点：点击地点弹出信息卡，显示 Wiki 信息、评分、评论与快捷操作（导航、收藏、分享）。
  - 图层切换：按类别（学术、生活、休闲、交通、行政等）切换显示图层。
- 校园 Wiki
  - 权威信息发布与管理：支持管理员审核用户提交的 Wiki 修改建议并记录变更历史。
- 用户社区
  - 评分与评论：1–5 星评分、图文评论、上传照片、标签、点赞、回复、举报。
  - 内容标签：记录标签热度并用于筛选。
  - 个人中心：我的评论/评分、收藏、浏览历史、消息通知等。
- 搜索与导航
  - 全局智能搜索与搜索建议（实时匹配、历史与热门关键词）。
  - 校内路径规划（步行/骑行）、一键导航、路线收藏。
- 后台管理
  - 内容审核、地点信息 CRUD、批量导入/导出、数据统计与可视化看板。
- 账号管理
  - 注册、登录、信息管理、登出、管理员封禁功能。
